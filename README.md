# Algorithmic_Extractors
This GitHub will contain the code for generating the DWT, GABOR and HOG images used in the article "Évaluation d’une Stratégie d’Hybridation Primitives-Transformeur pour Quantifier la Sévérité de Pneumonies" once it is public.

Once downloaded, the user will need to adapt the input and output paths as well as the parameters to chose the specific extractor variant required.
